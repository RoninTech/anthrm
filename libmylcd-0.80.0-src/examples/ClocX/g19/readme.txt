
An output plugin for ClocX - http://www.clocx.net


Copy lglcd.dll to the ClocX directory were ClocX.exe resides.
eg; c:\Program Files\ClocX\lglcd.dll

Copy g19plugin.dll to the ClocX plugins directory.
eg; c:\Program Files\ClocX\plugins\g19plugin.dll


You will need to restart ClocX for the plugin to appear in plugin list.
To enable put a tick beside g19plugin.dll in Clocx's plugins tab. Restart Clocx.
